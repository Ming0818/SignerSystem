This project includes three parts,
User Interface: using MFC to design simple but intuitive UI.
DataBase: choosing ODBC to manage the information about the user and singer we suppose. and we have implemented automatic regitration of the data source on the target machine.
Speaker Recognition algorithm: GMM + MFCC, implemeted by matlab due to complicated matrix operation.

In our design, we take the technology that the C++ code calls the Matlab dll.
